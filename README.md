# ViewBot for YouTube and Rumble

Works with both YouTube and Rumble platforms. Focuses on quality proxies, fast getting started, and clear installation guidance.

## Unparalleled Functionality
Experience a high level of functionality with the YouTube & Rumble Watch Bot. Equipped with advanced algorithms for view botting, customizable watch time, and engagement features, the program offers versatile automation to enhance your metrics and accelerate growth.

## User-Friendly Interface
Navigate a feature-rich interface designed to reduce complexity. Control workers, proxies, videos, and settings from an intuitive UI so you spend less time on setup and more time iterating on content strategy.

## Seamless YouTube and Rumble Integration
The app integrates with both platforms’ flows. It streamlines navigation, skips unnecessary prompts, and aligns with core features for a smooth workflow across YouTube and Rumble.

## Requirements
- Node.js 18.16.10 or above
- Stable connection: at least ~2 megabits per minute per worker
- Quality proxies (recommended) to maximize view distribution
- A capable computer to sustain concurrent workers

## Installation (Windows)
- Install Node.js LTS and Git.
- In the project root, run: `npm install`
- Setup: `npm run setup:win`
- Start (development): `npm start`
- Packaged build: `npm run package` then run the EXE in `out/`

## Installation (Linux/Mac)
- Install Node.js LTS and Git.
- In the project root, run: `npm install`
- Linux: `npm run setup:linux`
- macOS: ensure build tools and browser dependencies are available
- Start (development): `npm start`

## Getting Started
- Proxies: Open Proxies, paste one per line, click “Check proxies”; use the Good tab to select working ones.
- Videos: Open Videos, add YouTube/Rumble URLs, select desired watch time and optional likes/comments.
- Settings: Tune worker count, bandwidth preferences, retry behavior, and platform-specific options.
- Run: Use “Start workers” to begin and “Stop workers” to halt; monitor logs and stats.

## FAQs
- How do I add proxies?  
  Open Proxies, paste one per line, click “Check proxies”; working entries appear under Good.
- How do I add videos?  
  Open Videos, paste the URL, set watch time, select options, save.
- Do I need a key?  
  In this build, features are available for testing without keys.
- Why are views not increasing?  
  Verify proxy quality, worker count, and network stability; avoid rapid IP reuse.
- How do I import/export accounts?  
  Use import/export buttons on Videos for lists tied to specific videos.
- Livestreams and shorts?  
  Supported with dedicated modes; Rumble UI has variations in flow.
- Headless mode?  
  Use headless scripts or settings to reduce resource usage.
